# CMGT/559 – Managing Risk and Security vs. Opportunity

This folder contains completed labs for CMGT/559: Managing Risk and Security vs. Opportunity. This course focuses on the balance between enabling business innovation and enforcing cybersecurity risk management practices. Labs reflect practical application of strategic risk mitigation tools and threat visibility platforms.

---

## Labs Completed

| Lab Title                                                  | Description |
|-------------------------------------------------------------|-------------|
| Demonstrating Mapping and Quarantining a Vulnerable System  | Used network mapping and isolation techniques to identify and quarantine at-risk hosts, reinforcing containment policies within enterprise architecture |
| Performing Vulnerability Scan Using OSSIM                   | Leveraged OSSIM (Open Source Security Information Management) to identify system vulnerabilities and assess asset risk posture using SIEM correlation and vulnerability plugins |

---

## Key Concepts Covered

- Threat detection using open-source SIEM tools  
- Network-based risk mapping and containment  
- Asset identification and exposure classification  
- Vulnerability correlation and response prioritization  
- Governance and business continuity integration  
- Aligning mitigation strategies with enterprise goals
