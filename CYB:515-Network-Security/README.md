# CYB/515 – Advanced Network Defense and Threat Management

This folder contains all labs completed for CYB/515: Advanced Network Defense and Threat Management. The course focuses on real-world tools, attack simulation, and defense strategies used to identify, prevent, and mitigate cybersecurity threats in enterprise environments.

---

## Labs Completed

| Lab Title                          | Description |
|-----------------------------------|-------------|
| Lab 1 – Using EtherDetect          | Captured and analyzed raw network traffic to identify potential unencrypted data exposure in real time |
| Lab 2 – Cracking Passwords with Ophcrack | Used rainbow tables to recover Windows passwords and evaluate weaknesses in unsalted hash storage |
| Lab 3 – ShareEnum                  | Enumerated open file shares on a Windows network to detect insecure or overly permissive configurations |
| Lab 4 – Using MBSA                 | Scanned a Windows system for missing updates and configuration risks using Microsoft Baseline Security Analyzer |
| Lab 5 – Advanced IP Scanner        | Identified live hosts, open ports, and exposed services on a LAN using a simple but effective scanning tool |
| Lab 6 – Simulating a Remote Access Trojan | Created a benign RAT in a lab environment to understand remote control risks and detection challenges |
| Lab 7 – Creating a Security Policy | Configured local security policy in Windows to enforce password complexity, auditing, and user rights restrictions |
| Lab 8 – Using eLiteWrap            | Explored executable wrapping techniques and the implications for software deployment and malware concealment |
| Lab 9 – Using John the Ripper      | Cracked password hashes using dictionary and brute-force techniques to demonstrate password storage weaknesses |
| Lab 10 – Rainbow Tables            | Used precomputed hash chains to recover plaintext passwords from unsalted hashes, emphasizing need for salting |

---

## Key Concepts Covered

- Network surveillance and traffic visibility
- Windows enumeration and configuration auditing
- Password cracking techniques and prevention
- Hash vulnerabilities and storage best practices
- Security policy enforcement at the OS level
- Executable packaging and detection
- Remote access risk awareness
- Defensive monitoring and policy hardening
