# CYB/530 – Cybersecurity Practitioner

This folder contains lab documentation and technical exercises completed for the Cybersecurity Practitioner course. The labs focus on hands-on skills essential for operational security roles, including real-world network analysis, secure system configuration, protocol-level inspection, and vulnerability remediation.

Each lab is structured using a consistent format that includes objective, tools used, procedure, results, and lessons learned. This approach supports reproducibility, audit readiness, and professional presentation of practical experience.

---

## Topics Covered

- Live packet capture and analysis using `tcpdump` and Wireshark  
- MAC address spoofing using SMAC  
- Client-server traffic inspection and protocol behavior  
- SSH session configuration from Windows to Windows Server  
- Installing and configuring IIS Web Server  
- Workstation vulnerability remediation  
- Network-wide vulnerability scanning and patching  
- DNS security through prevention of unauthorized zone transfers  

---

## Tools Utilized

- Kali Linux and Windows 10/Server  
- Wireshark, tcpdump, SMAC, Nessus, OpenVAS  
- PowerShell, Command Prompt, Group Policy  
- BIND DNS, IIS, SSH, NSLookup, dig  

---

## Purpose

The purpose of this course and its accompanying labs is to simulate real-world cybersecurity practitioner responsibilities. Each exercise demonstrates a skill that contributes to the detection, mitigation, or prevention of security risks across systems and networks.

This folder contributes to a larger cybersecurity portfolio that showcases my technical ability, problem-solving, and adherence to professional security practices.
