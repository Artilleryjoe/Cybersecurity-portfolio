# CYB/510 – Cloud Security

This folder contains labs completed for CYB/510: Cloud Security. The course focused on securing enterprise systems through hands-on experience with virtualization, VPNs, intrusion detection, cryptographic tools, and network attack simulations.

All labs were conducted in a secure and controlled environment, and all offensive techniques were used strictly for academic and educational purposes.

---

## Labs Completed

| Title                                                      | Description |
|-------------------------------------------------------------|-------------|
| Creating a DMZ Using GNS3 and PuTTY                         | Designed and configured a network topology with a DMZ using GNS3 and remote CLI access via PuTTY |
| Creating a VM and Installing Ubuntu Using Hyper-V           | Built a secure lab environment by creating a virtual machine and installing Ubuntu in Hyper-V |
| Building an IPsec VPN                                       | Configured a site-to-site VPN using strongSwan to protect communications across untrusted networks |
| Configuring RAID 5                                          | Created a software-based RAID 5 array using `mdadm` to improve redundancy and fault tolerance |
| Observing a SHA-Generated Hash Value                        | Generated and analyzed SHA-1, SHA-256, and SHA-512 hash values to verify file integrity |
| Performing a DoS Attack Using a SYN Flood                   | Simulated a SYN flood attack to study TCP exhaustion and system response under denial-of-service conditions |
| Using SEToolkit to Plan a Social Engineering Attack         | Simulated a phishing attack with SEToolkit to analyze the risks of cloned login pages in training contexts |
| Performing a Credential-Based Brute Force Attack            | Used Hydra to simulate brute force attacks on SSH and HTTP logins, testing defense mechanisms like logging and fail2ban |

---

## Key Skills Demonstrated

- Network segmentation and firewall configuration  
- Secure system deployment with virtualization tools  
- VPN tunneling and IPsec encryption  
- RAID implementation and disk-level resilience  
- Cryptographic hashing for data integrity  
- Ethical simulation of DoS and credential attacks  
- Social engineering awareness and web-based phishing simulation  
- Intrusion detection techniques and countermeasure validation
