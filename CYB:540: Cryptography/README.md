# CYB/540 – Cryptography

This folder contains labs exploring fundamental and applied cryptographic techniques, including symmetric and asymmetric encryption, steganography, key management, and security infrastructure. Each lab is performed in a controlled environment and documented in present tense to reflect active, hands-on implementation.

---

## Topics Covered

- Covert communication using steganography  
- Risks and implications of keylogging  
- Infrastructure preparation for high availability  
- RSA key pair generation and usage  
- Asymmetric encryption for secure data exchange  

---

## Completed Labs

- Using Steganography  
- Enabling a Keylogger on a Target Machine  
- Installing the Failover Cluster Feature  
- OpenSSL Private/Public Key Pair Generation  
- Using RSA Asymmetric Algorithm  

---

## Tools and Technologies

- OpenSSL  
- steghide  
- Python (`pynput` for keylogging)  
- Windows Server (Failover Clustering)  
- Linux terminal / PowerShell  
- Text files and PEM-formatted key files  

---

## Purpose

These labs reinforce cryptographic principles with real-world tools and practical scenarios. By executing and understanding both secure and insecure practices, students gain deeper insight into confidentiality, integrity, and authentication. The course emphasizes secure key management and the critical importance of cryptography in modern cybersecurity.
