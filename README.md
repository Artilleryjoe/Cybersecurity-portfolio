About This Repository

This GitHub repository serves as a comprehensive, course-aligned portfolio of practical cybersecurity labs, assessments, and applied research completed throughout the Master of Science in Cybersecurity program. Each folder represents a course, containing individually documented labs or projects formatted for professional readability.

All content has been ethically executed in isolated or simulated environments and is intended for educational and demonstrative purposes only.

Courses and Projects

CYB/500 – Advanced Cybersecurity Concepts

Labs include MITM attacks, memory-based buffer overflows, Snort IDS configuration, session hijacking, and XSS attacks

CYB/505 – Secure Software Development

Focused on the SDLC with applied secure coding practices and software lifecycle models

CYB/510 – Enterprise Network Security

Topics include VM installation, IPSec VPN setup, RAID configuration, reverse shells, RAT deployment, and system hardening

CYB/515 – Foundations of Cybersecurity Operations

Covered traffic analysis, web server configuration, vulnerability remediation, and secure shell sessions

CMGT/559 – Managing Risk and Security vs. Opportunity

Research-based deliverables focusing on risk mitigation, threat intelligence, and governance frameworks

CYB/530 – Cybersecurity Practitioner

Involved packet analysis, MAC spoofing, SSH sessions, web server installations, DNS hardening, and vulnerability response

CYB/535 – Secure Programming

Labs included logging and monitoring, unit testing, fixing XSS and password hash vulnerabilities, and securing data in transit and at rest

CYB/540 – Cryptography

Explored steganography, RSA encryption, key generation, failover clustering, and keystroke logging

CYB/550 – Technical Enterprise Security

Mixed labs and presentation work on Exif metadata, CSRF attacks, ARP poisoning, Netcat reverse shells, and OSINT case studies

CYB/555 – Cybersecurity Architecture

Delivered practical labs on AD DS setup, DNS zone transfers, enterprise-level architecture diagrams, and risk management for technical stakeholders

CYB/560 – MS Cybersecurity Capstone

Final course with advanced labs on Vega vulnerability scanning, TOCTOU exploitation, DHCP starvation, Autopsy forensics, and risk-mitigation presentations
