# CYB/550 – Technical Enterprise Security

## Course Overview  
This course focuses on identifying, evaluating, and mitigating enterprise-level security threats through hands-on labs and applied cybersecurity techniques. Topics covered include open-source intelligence (OSINT), exploitation techniques, secure system architecture, and operational defenses. Learners are introduced to both technical tools and strategic concepts used in enterprise environments.

## Labs Completed

### Using ExifTool for Metadata Extraction  
Demonstrated how to analyze image metadata for privacy and OSINT risks.

### Conducting a CSRF Attack  
Simulated a cross-site request forgery against a vulnerable web app (DVWA) to demonstrate session-based exploitation.

### Performing ARP Poisoning  
Executed a man-in-the-middle attack using ARP spoofing and Wireshark to capture intercepted data.

### Bypassing Command Shell Restrictions  
Escaped a restricted Linux shell to gain full command-line access, simulating sandbox evasion.

### Using Netcat for Reverse Shell  
Created a reverse shell from a target machine to gain remote access and simulate post-exploitation techniques.
