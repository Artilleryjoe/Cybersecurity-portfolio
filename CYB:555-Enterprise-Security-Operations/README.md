# CYB/555 – Cybersecurity Architecture

## Course Overview  
This course explores the principles, practices, and technologies involved in designing secure enterprise environments. Students learn to align cybersecurity architecture with business goals and risk tolerance, while applying real-world tools and frameworks.

---

## Organization Profile  
**Company Name**: Sicherheitsschaf GmbH  
**Location**: Berlin, Germany  
**Motto**: “Schafe zählen, Sicherheit gewinnen!” (“Count sheep, gain security!”)  
**Focus**: Enterprise cybersecurity consulting and managed security services  

Key Services:  
- Secure network and domain architecture  
- Mobile device security and virtualization  
- Policy development and framework alignment  
- Cybersecurity assessments and training

## Week 4 Deliverable: Mobile Device Security Plan

**Title**: Internal Report: Recommendations for Securing Mobile Device Usage  
**Highlights**:
- IT-managed procurement and baseline security configuration  
- Role-based access control  
- Encryption standards (FDE, E2EE)  
- MDM policies, remote wipe, secure VPN  
- Aligned with BSI TR-03139 and NIST guidance
