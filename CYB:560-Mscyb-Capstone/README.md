# CYB/560 – MS Cybersecurity Capstone

## Course Overview
This final capstone course represents the culmination of all prior cybersecurity training and applied knowledge throughout the Master’s program. Labs, projects, and deliverables reflect both offensive and defensive capabilities, incident response proficiency, and cybersecurity strategy at the enterprise level.

---

## Key Labs and Activities

### Lab Work
- **Footprinting a Website using Maltego**  
  Mapped external web assets, domains, and infrastructure using OSINT.

- **Using OSQuery for Incident Response**  
  Queried live system data to detect anomalies and lateral movement patterns.

- **Analyzing Forensic Data with Autopsy**  
  Conducted disk and memory analysis using forensic tools to recover deleted files and trace user behavior.

- **Exploiting TOCTOU Vulnerability in Web Apps**  
  Demonstrated race condition exploitation during file permission checks.

- **Configuring an SSH Server**  
  Hardened and verified secure remote access through SSH on a Linux system.

- **Resetting the Root Password**  
  Performed secure recovery techniques using bootloader and live system manipulation.

- **DHCP Starvation Attack Simulation**  
  Deployed a rogue DHCP server to test the effectiveness of DHCP snooping defenses.

- **Web App Security Analysis using Vega**  
  Scanned for XSS, SQLi, and insecure configurations on a live test environment.

---

## Capstone Deliverables

- **Week 3 Mitigation Presentation**  
  Delivered strategic mitigation recommendations for high-risk vulnerabilities.

- **Week 6 Risk Management Presentation**  
  Provided a board-facing security proposal based on current risk posture and threat landscape.

## Lessons Learned

- Integrated tools like OSQuery, Autopsy, and Vega allow for precise threat detection and remediation.
- Offensive security techniques (e.g., TOCTOU, DHCP starvation) must be counterbalanced with defense-in-depth strategies.
- Effective reporting and risk communication is as important as technical mitigation.
- The capstone reinforced both my hands-on technical growth and my ability to communicate findings in business and security terms.
