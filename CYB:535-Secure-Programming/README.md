# CYB/535 – Secure Programming

This folder contains secure programming lab work focused on identifying, simulating, and remediating common software vulnerabilities. All labs are developed using Python and executed in PyCharm, with an emphasis on present-tense procedural documentation and real-world security practices.

---

## Topics Covered

- Logging and monitoring deployed applications  
- Automating unit tests with `unittest`  
- Fixing authentication and role-based access control defects  
- Implementing encryption for data in transit and at rest  
- Demonstrating a persisted XSS attack in an administrative interface  

---

## Completed Labs

- Monitoring and Logging a Deployed Application  
- Using a Test Suite to Automate Unit Testing  
- Handling Authentication and Authorization Defects  
- Protecting Data in Transit and Data at Rest  
- Staging a Persisted XSS Attack on an Admin Function  
- (In Progress) Cracking a Password Hash  
- (In Progress) Fixing a Password Hash Vulnerability

## Purpose

This course develops a secure-by-design mindset through hands-on experimentation with code flaws and cryptographic protections. The labs simulate software-level vulnerabilities and reinforce the value of proactive coding techniques, validation, and testing to protect end users and infrastructure.
